# SentinelBox Real-time Dashboard

A React-based dashboard for monitoring AI Agent Sandboxes.

## Features
- Real-time CPU/Memory telemetry using eBPF streaming.
- Audit logs for security signals (SIGSYS, EPERM) and system events.

## Tech Stack
- React + Vite
- Tailwind CSS v4
- Recharts (Time-series data visualization)
- Lucide React (Icons)
